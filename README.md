# StayEase: Airbnb Clone Project

StayEase is a full-stack Airbnb clone web application. It allows users to search, view, and book vacation properties. This project focuses on building a responsive, accessible, and user-friendly accommodation booking platform that mimics Airbnb‚Äôs core functionalities.

- Prioritize accessibility and mobile-first design.
